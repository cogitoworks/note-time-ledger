# Chrome Web Store 掲載用テキスト

## 拡張機能名
note Time Ledger

## 短い説明（132文字以内）
noteダッシュボードの記事数値に時間軸を与える観測ツール。ビュー・スキ・コメントを日次で記録し、記事の伸び方を見える化します。データは完全ローカル保存。

## 詳細説明

noteのアクセス状況ページに表示される記事ごとの数値を、ボタン一押しでブラウザ内に記録します。

■ できること
・ビュー数、スキ数、コメント数を日次で記録
・記事ごとの現在値と前回からの変化量を一覧表示
・記事をクリックして日次の履歴を確認
・JSON形式でのデータエクスポート/インポート（バックアップ・復元）

■ できないこと（意図的に除外しています）
・バズ予測やAI分析
・読者ペルソナの推定
・記事テーマの提案
・他ユーザーとの比較
・外部サーバーへのデータ送信

■ プライバシー
・データはすべてブラウザ内（IndexedDB）に保存されます
・外部サーバーへの通信は一切ありません
・noteのサーバーに追加のリクエストを送信しません
・ページ上にすでに表示されている数値を読み取るだけです

■ 使い方
1. noteにログインし、アクセス状況ページ（note.com/sitesettings/stats）を開く
2. ページ右下の「📊 記録する」ボタンをクリック
3. ツールバーのアイコンをクリックして一覧を確認

■ 注意事項
・note株式会社が提供する公式ツールではありません
・noteのUI変更により動作しなくなる場合があります
